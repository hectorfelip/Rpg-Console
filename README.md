# Rpg-Console
Treino de lógica de programação em java

Caso modifique o código mande no console:\n
  javac -cp "lib\gson-2.10.1.jar" -d out Jogador.java JogadorUse.java Main.java Personagem.java projectRepo.java\n
    Caso adicione um arquivo novo adicione ele nessa linha de comando:\n
      javac -cp "lib\gson-2.10.1.jar" -d out Jogador.java JogadorUse.java Main.java Personagem.java projectRepo.java novoArquivo.java\n

Para executar:\n
  java  -cp "out;lib\gson-2.10.1.jar" Main
